**"Where Velvet Meets Radiance"**

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd veloura
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:3000`

### Build for Production

```bash
npm run build
npm start
```
